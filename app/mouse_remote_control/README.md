# 鼠标的远程控制：可用于反向控制

```
                                                                                           +------------+
                        +-------------+                           +--------------+         |            |
                        |  开发板A    | +-----------------------> |   开发板B    |   USB   |            |
+-------+     USB       |             |        WIFI net           |              +--------->    PC      |
| mouse +--------------->    R        | <-----------------------+ |     S        |         |            |
+-------+               |             |                           |              |         |            |
                        +-------------+                           +--------------+         |            |
                                                                                           +------------+
```

R/S之间使用裸数据进行透传。

## R

`USB host`枚举出USB设备后，通过`/dev/hidraw[0-9]`进行操作。

1. 读取`dev/hidraw[0-9]`节点的数据
2. 网络传输，写


## S

`USB device`模拟一个USB设备`/dev/hidg[0-9]`，如鼠标。

1. 网络传输，读
2. 将读到的数据写入`/dev/hidg[0-9]`
